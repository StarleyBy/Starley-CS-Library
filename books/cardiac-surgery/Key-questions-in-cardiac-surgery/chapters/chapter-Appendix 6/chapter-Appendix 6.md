## **Appendix VI**

## Normal cardiac physiological values

|                                                                         | Abbreviation | Formula                     | Normal range              |
|-------------------------------------------------------------------------|--------------|-----------------------------|---------------------------|
| Systolic blood pressure                                                 | SBP          |                             | 90-140mmHg                |
| Diastolic blood pressure                                                | DBP          |                             | 60-90mmHg                 |
| Mean arterial blood pressure                                            | MAP          | [(SBP - DBP)/3]<br>+ DBP    | 70-105mmHg                |
| Left atrial pressure                                                    | LAP          |                             | 6-12mmHg                  |
| Central venous (right atrial) pressure CVP                              |              |                             | 2-6mmHg                   |
| Arterial oxygen saturation                                              | SaO2         |                             | 95-100%                   |
| Mixed venous saturation                                                 | SvO2         |                             | 60-80%                    |
| Pulmonary artery systolic pressure                                      | PASP         |                             | 15-30mmHg                 |
| Pulmonary artery diastolic pressure                                     | PADP         |                             | 5-15mmHg                  |
| Mean pulmonary artery pressure                                          | MPAP         | [(PASP - PADP)/3]<br>+ PADP | 10-20mmHg                 |
| Pulmonary artery wedge pressure                                         | PAWP         |                             | 6-12mmHg                  |
| Systemic vascular resistance                                            | SVR          | 80 x (MAP - CVP)<br>/ CO    | dyne/sec/cm5     |
| Systemic vascular resistance index                                      | SVRI         | SVR / BSA                   | dyne/sec/cm5/m2 |
| Pulmonary vascular resistance                                           | PVR          | 80 x (MPAP - PAWP)<br>/ CO  | dyne/sec/cm5      |
| Pulmonary vascular resistance index PVRI                                |              | PVR / BSA                   | dyne/sec/cm5/m2   |
| Heart rate                                                              | HR           |                             | bpm                |
| Stroke volume                                                           | SV           |                             | 70-90mL (1mL/kg)          |
| Cardiac output                                                          | CO           | SV X HR                     | 4-8L/min                  |
| Cardiac index                                                           | CI           | CO / BSA                    | 2.5-4L/min/m2             |
| for resistances, to convert into woods units, divide dyne/sec/cm5 by 80 |              |                             |                           |
